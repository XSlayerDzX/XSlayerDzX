## 🏆 Featured Project — ArenaBrain

> An autonomous Clash Royale agent + analytics web platform

Built with my teammate [Mousli Tayeb](https://github.com/mousli-tayeb), supervised by **Prof. Belkacem Khaldi**.

**How it works:**
- 🎮 Captures the game screen in real time via BlueStacks
- 👁️ Detects units, towers, and cards using **RF-DETR**
- 🧠 Makes decisions through an **LSTM policy** trained with behavior cloning → fine-tuned with **PPO**
- 📊 Sends training logs and stats to the **ArenaBrain dashboard** via API key

The PPO stage shows promising results — it just needs more compute and training time to fully shine. That chapter is still open.
